Política de privacidad

Última actualización: 15/02/2026

Responsable

Este sitio web (aprendersql.com) es un curso gratuito de SQL.

Datos que recopilamos

Este sitio es estático y no recopila datos personales de forma activa. No hay formularios de registro, ni cuentas de usuario, ni almacenamiento de tu nombre o correo.

Cookies

Este sitio no utiliza cookies propias para identificarte o seguir tu actividad. Si en el futuro se incorporan (por ejemplo, para recordar preferencias), se indicará aquí.

Enlaces a otros sitios

El curso puede incluir enlaces a documentación o recursos externos. No somos responsables de las prácticas de privacidad de esos sitios.

Cambios

Podemos actualizar esta política de privacidad ocasionalmente. La fecha de "Última actualización" al inicio del documento reflejará los cambios relevantes.

Aceptación

El uso de este sitio implica la aceptación de esta política de privacidad.
